How to serialize out:
    SerializeOut(node, key, value)
    
    If we want to serialize a struct:
        child = node.append_child(key)
            SerializeOut(child, "member", mMember)
            ...

How to serialize in:
    SerializeIn(node, value)
    
    If we want to serialize a struct:
        SerializeIn( node.child("member") , mMember)
        ...

    PODs
